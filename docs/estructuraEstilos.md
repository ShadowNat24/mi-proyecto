# Guía de Estilos - Proyecto Todo en Blanco

Este documento detalla la identidad visual y las reglas de diseño aplicadas en el sitio web para mantener la consistencia en todas las páginas.

## 1. Paleta de Colores
Se utilizaron colores que transmiten seriedad y confianza, alineados con la identidad de la empresa logística:
* **Azul Profesional (#47606f):** Color principal para encabezados, menús de navegación y botones de acción.
* **Amarillo Vial (#eeb906):** Color de acento utilizado para resaltar elementos importantes y estados "hover".
* **Blanco (#ffffff) y Gris Claro (#f4f4f4):** Utilizados para fondos y áreas de lectura para mantener la claridad.

## 2. Tipografía
* **Fuente Principal:** 'Montserrat', sans-serif. 
* **Origen:** Google Fonts.
* **Uso:** Se aplica en todo el sitio para dar una estética moderna y legible.

## 3. Componentes Unificados
* **Navegación:** Menú horizontal con lista desordenada (`<ul>`), sin viñetas y con distribución `flex`.
* **Botones:** Bordes redondeados (`border-radius: 5px`), con transiciones suaves de color al pasar el mouse.
* **Formularios:** Etiquetas arriba de los inputs, espaciado con `padding` para facilitar la escritura y validaciones visuales.

## 4. Estructura de Archivos CSS
* Los estilos globales se encuentran en: `css/estilo.css`.
